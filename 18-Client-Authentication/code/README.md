# Chapter 18 Code & Labs

Thư mục này cung cấp môi trường thực hành Client Authentication Oanh Cáp Trọng Lõi Tự Trị.

## Khởi động môi trường

1. Mở terminal tại thư mục này.
2. Chạy lệnh:
```bash
docker-compose up -d
```
3. Đợi vài giây, truy cập `http://localhost:8080` (admin/admin).

## Hướng dẫn thực hành (Labs)
Mở file `../Labs/Lab-1-Exercises.md` để xem hướng dẫn thực hành Mô Phỏng Dòng Mạch Ngầm Oanh Giao Dịch Client Bằng Postman Lệnh Đáy Oanh Mạch Rút Trọng.

## Dọn dẹp môi trường

```bash
docker-compose down -v
```
