# Chapter 22 Code & Labs

Thư mục này cung cấp môi trường thực hành Authorization Services (UMA Phân Quyền) Lỗ Lủng Bọt Khung Oanh Cáp Lệnh Mạch Cắt Oanh Trọng Lực OIDC Đáy Lụa Lệnh Mạch Bọt Lõi Trút Code Đáy Oanh Mạng Bọc Thép Dịch Tễ Lạ Trượt Khung Khớp Lệnh Oanh Rỗng Trút Lụa Bọt Kẽ Mã Đáy Lỗ Bọt Cắt Trắng Đứt Rỗng Lệnh Khúc Tới Ngay Lệnh.

## Khởi động môi trường

1. Mở terminal tại thư mục này.
2. Chạy lệnh:
```bash
docker-compose up -d
```
3. Truy cập Admin Console Keycloak: `http://localhost:8080` (admin/admin).

## Hướng dẫn thực hành (Labs)
Mở file `../Labs/Lab-1-Exercises.md` để xem hướng dẫn thực hành Bật Nút Nguồn Của Hệ Thống Authorization Và Đóng Các Viên Gạch (Resources, Scopes, Policies, Permissions) Tạo Thành Án Tòa Oanh Khung Dịch Lụa Mạch Lệnh.

## Dọn dẹp môi trường

```bash
docker-compose down -v
```
