# Chapter 17 Code & Labs

Thư mục này cung cấp môi trường thực hành Giao Thức SAML bằng Keycloak.

## Khởi động môi trường

1. Mở terminal tại thư mục này.
2. Chạy lệnh:
```bash
docker-compose up -d
```
3. Đợi vài giây, truy cập `http://localhost:8080` (admin/admin).

## Hướng dẫn thực hành (Labs)
Mở file `../Labs/Lab-1-Exercises.md` để xem hướng dẫn thực hành Đóng Vai Lập Trình Viên Enterprise Nhập Khẩu Metadata Nối Kết 2 Máy Chủ Bằng XML Chữ Ký Lụa Oanh Mạng Bọc Thép Dịch Tễ Lạ!

## Dọn dẹp môi trường

```bash
docker-compose down -v
```
