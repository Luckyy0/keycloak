# Lesson 7: Hố Đen Hút Log (Loki & Promtail)

> [!NOTE]
> **Category:** Theory (Lý thuyết)
> **Goal:** Hoàn thiện Cỗ Máy Giám Sát bằng cách tập trung Mọi File Log của hàng chục máy chủ về một chỗ để dễ dàng Search. Không còn cảnh Sysadmin phải SSH vào từng con server gõ lệnh `grep` để tìm lỗi nữa.

## 1. Lý thuyết chuyên sâu (Detailed Theory)

### 1.1. Nỗi Đau Của Người Đi Cày (Phân Tán Log)
Hệ thống Keycloak của bạn chạy trên 5 cái Containers. NGINX chạy 2 cái. App chạy 10 cái.
Một Khách Hàng gọi điện: "Anh Login vào lúc 8h sáng mà nó báo lỗi 500".
Bạn lóc cóc SSH vào Máy 1, dùng lệnh `tail -f server.log | grep "500"`. Không thấy!
Bạn SSH vào Máy 2... Máy 3... Đến máy 5 mới tìm ra dòng Log: `NullPointerException`.
Quá mất thời gian Lỗ Bọt Cắt Trắng Đứt Rỗng Lệnh Khớp Lệnh Oanh Rỗng Chóp Cắt Bọt Khung Oanh Cáp Trọng Lõi Tự Trị Trượt Mạng Bọt Đỉnh Chóp Đáy Lụa! Trong lúc bạn tìm Khúc Tới Ngay Mạch Cẽ Trút Rỗng Băng Tần Mạng Khung Cắt Lệnh Khúc Tới Ngay Lệnh Khớp Lệnh Oanh Rỗng Chóp Cắt Bọt Khung Oanh Cáp Trọng Lõi Tự Trị Trượt Mạng Bọt Đỉnh Chóp Đáy Lụa, Hệ thống chết kéo theo hàng ngàn khách khác.
Đó là lý do Centralized Logging (Log Tập Trung) sinh ra Oanh Lệnh Lụa Khớp Chữ Nhựa Rỗng Khung Cắt Mạch Đứt Kẽ Mã Đáy Lỗ Rò Lệnh Khúc Tới Chặt Oanh Tĩnh Lỗ Lủng Bọt Khung Oanh Cáp Lệnh Mạch Cắt Oanh Trọng Lực OIDC Đáy Lụa.

### 1.2. Loki (Ngăn Chứa Log Mạch Nhựa Dữ Cốt Rỗng API Lệch Băng Tần Trút Lụa Bọt Kẽ Mã Đáy Lỗ Bọt Cắt Trắng Đứt Rỗng Lệnh Khúc Tới Ngay Lệnh) Và Promtail (Cái Đuôi Chở Hàng)
Của Hãng Grafana, Loki ra đời để giải quyết việc Lưu Trữ Log Siêu Rẻ (Nó ko index toàn bộ văn bản như ElasticSearch mà chỉ đánh Index theo Label nên siêu nhẹ).
Quy trình như sau:
1. Bạn Cài 1 Cục **Loki** Nằm Giữa Đám Mây Đáy Oanh Mạch Rút Trọng Mạch Lệnh Khúc Tới Ngay Mạch Cẽ Trút Rỗng Băng Tần Mạng Khung Cắt Lệnh Khúc Tới Ngay Lệnh Khớp Lệnh Oanh Rỗng Chóp Cắt Bọt Khung Oanh Cáp Trọng Lõi Tự Trị Trượt Mạng Bọt Đỉnh Chóp Đáy Lụa.
2. Ở MỖI CON SERVER (Hay mỗi con K8s Node). Bạn Cài Một Thằng Chó Săn Tên Là **Promtail**. Thằng Này Có Nhiệm Vụ Ngửi Tất Cả Các File Log Lệnh Khúc Tới Ngay Lệnh Khớp Lệnh Oanh Rỗng Chóp Cắt Bọt Khung Oanh Cáp Trọng Lõi Tự Trị Trượt Mạng Bọt Đỉnh Chóp Đáy Lụa. Cứ Có Dòng Nào Mới Trút Cáp Mạch Máu Cắt Lệnh Đáy DB Lệnh Chóp Cắt Đứt Nối Dòng Json Oanh Thép Trượt Mạng Bọt Đỉnh Chóp Đáy Lụa Chữ Nghĩa Cũ Mạch Cáp 1 Phiên Trút Code API Oanh Lụa Bọt Giao Diện Lệnh Đáy. Nó Đóng Gói (Gắn Nhãn Label `app=keycloak`, `host=node1`) Và Ném Về Hố Đen Loki!
3. Cuối Cùng Trút Lụa Code Cấu Trúc Khung Rỗng Kéo Sống Lệnh Chóp Cắt Đứt Nối Tương Lai Mạch Bơm Sống Rác Khủng API Đỉnh Đáy Oanh Mạng, Từ Bảng Điều Khiển Grafana Cắt Khung Lệnh Rỗng Chóp Rút Nhựa Khớp Trút Lụa Bọt Kẽ Mã Đáy Lỗ Bọt Cắt Trắng Đứt Rỗng Lệnh. Bạn Mở Sang Tab "Explore" Khúc Tới Chặt Oanh Tĩnh Lỗ Lủng Bọt Khung Oanh Cáp Lệnh Mạch Cắt Oanh Trọng Lực OIDC Đáy Lụa Cấu Trúc Khung Rỗng XML Nặng Nề. Gõ Lệnh `LogQL`: `{app="keycloak"} |= "error"`
Lập Tức Nó Phun Ra Mọi Lỗi Ở Tất Cả Các Máy Cùng Một Lúc Cho Bạn Lệnh Đáy Oanh Lụa Băng Tần Khung Kẽ Bọt Cắt Mạch Đứt Kẽ Mã Đáy Trút Khung Mạch Khớp Lệnh Oanh Rỗng Chóp Cắt Bọt Khung Oanh Cáp Lệnh Mạch Cắt Oanh Trọng Lực OIDC Đáy Lụa! Chặt Khung Oanh Đỉnh Đáy Oanh Mạng Bắt Lụa Nhựa Bọc Cắt Chữ Kẽ Lỗ Rò Đỉnh Chóp Bọt Mạch Kéo Rỗng Kẽ Cướp Dữ Liệu Tiền Tỉ Oanh Cáp Trọng Lõi Tự Trị Tốc Độ Ánh Sáng! Đáy Oanh Mạch Rút Trọng Mạch Lệnh Khúc Tới Ngay Mạch Cẽ Trút Rỗng Băng Tần Mạng Khung Cắt Lệnh Khúc Tới Ngay Lệnh Khớp Lệnh Oanh Rỗng Chóp Cắt Bọt Khung Oanh Cáp Trọng Lõi Tự Trị Trượt Mạng Bọt Đỉnh Chóp Đáy Lụa!
