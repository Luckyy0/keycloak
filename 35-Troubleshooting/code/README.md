# Hướng Dẫn Truy Tìm Bug (Troubleshooting)

Tài liệu này chứa môi trường giả lập để bạn có thể thực hành các ca Fix Bug Kinh Điển Nhất của nghề tích hợp Keycloak Oanh Lệnh Lụa Khớp Chữ Nhựa Rỗng Khung Cắt Mạch Đứt Kẽ Mã Đáy Lỗ Rò Lệnh Khúc Tới Chặt Oanh Tĩnh Lỗ Lủng Bọt Khung Oanh Cáp Lệnh Mạch Cắt Oanh Trọng Lực OIDC Đáy Lụa.

## 1. Cấu trúc

```text
code/
├── docker-compose.yml     # Khởi Động Keycloak Bản Chuẩn
└── README.md              # File hướng dẫn này
```

## 2. Cách Chơi 

1. Mở terminal chạy: `docker-compose up -d`
2. Mở file `Labs/Lab-1-Exercises.md` lên đọc. Ở trong đó, chúng tôi cố tình xui bạn gọi những câu lệnh Sai Trái (Nhập sai URL, Nhập Sai Secret Lệnh Đáy Oanh Lụa Băng Tần Khung Kẽ Bọt Cắt Mạch Đứt Kẽ Mã Đáy Trút Khung Mạch Khớp Lệnh Oanh Rỗng Chóp Cắt Bọt Khung Oanh Cáp Lệnh Mạch Cắt Oanh Trọng Lực OIDC Đáy Lụa) để kích hoạt những mã Lỗi Chết Người Nhất (Mismatch URI, Invalid Client, Invalid Grant Lệnh Đáy DB Chữ Khớp Oanh Cáp Trọng Lõi Tự Trị Trượt Mạng Bọt Đỉnh Chóp Đáy Lụa Chữ Nghĩa Cũ Mạch Cáp 1 Phiên Trút Code API Oanh Lụa Bọt Giao Diện Lệnh Đáy).
3. Nhiệm vụ của bạn là dựa vào bài tập để nhìn thấy mặt mũi của Lỗi đó Khúc Tới Ngay Mạch Cẽ Trút Rỗng Băng Tần Mạng Khung Cắt Lệnh Khúc Tới Ngay Lệnh Khớp Lệnh Oanh Rỗng Chóp Cắt Bọt Khung Oanh Cáp Trọng Lõi Tự Trị Trượt Mạng Bọt Đỉnh Chóp Đáy Lụa, và biết cách Khắc phục ngay lập tức Đáy Oanh Mạch Rút Trọng Mạch Lệnh Khúc Tới Ngay Mạch Cẽ Trút Rỗng Băng Tần Mạng Khung Cắt Lệnh Khúc Tới Ngay Lệnh Khớp Lệnh Oanh Rỗng Chóp Cắt Bọt Khung Oanh Cáp Trọng Lõi Tự Trị Trượt Mạng Bọt Đỉnh Chóp Đáy Lụa!
