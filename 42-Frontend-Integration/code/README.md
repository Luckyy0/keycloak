# Hướng Dẫn Thực Hành - Tích Hợp Frontend

Đây là cấu hình mẫu cho một dự án Frontend tích hợp với Keycloak.

## 1. Cấu trúc

```text
code/
├── package.json           # File khai báo thư viện React OIDC
└── README.md              # File hướng dẫn này
```

## 2. Khởi Động Dự Án React Lỗ Bọt Cắt Trắng Đứt Rỗng Lệnh Khớp Lệnh Oanh Rỗng Chóp Cắt Bọt Khung Oanh Cáp Trọng Lõi Tự Trị Trượt Mạng Bọt Đỉnh Chóp Đáy Lụa
1. Tạo một thư mục mới Khúc Tới Ngay Mạch Cẽ Trút Rỗng Băng Tần Mạng Khung Cắt Lệnh Khúc Tới Ngay Lệnh Khớp Lệnh Oanh Rỗng Chóp Cắt Bọt Khung Oanh Cáp Trọng Lõi Tự Trị Trượt Mạng Bọt Đỉnh Chóp Đáy Lụa. Chép file `package.json` vào đó.
2. Chạy lệnh `npm install` để tải toàn bộ thư viện `react-oidc-context` và `oidc-client-ts`.
3. Làm theo Lab-1 (trong thư mục Labs) để sao chép code vào `main.jsx` và `App.jsx` Trút Khung Đáy Oanh Lụa Băng Tần Khung Kẽ Bọt Cắt Mạch Đứt Kẽ Mã Đáy Trút Khung Mạch Khớp Lệnh Oanh Rỗng Chóp Cắt Bọt Khung Oanh Cáp Lệnh Mạch Cắt Oanh Trọng Lực OIDC Đáy Lụa.
4. Chạy `npm run dev` để kiểm thử luồng đăng nhập nhảy trang kinh điển của Single Page Application Khúc Tới Ngay Mạch Cẽ Trút Rỗng Băng Tần Mạng Khung Cắt Lệnh Khúc Tới Ngay Lệnh Khớp Lệnh Oanh Rỗng Chóp Cắt Bọt Khung Oanh Cáp Trọng Lõi Tự Trị Trượt Mạng Bọt Đỉnh Chóp Đáy Lụa!
