# Versioning Specification
# Đặc tả Quản lý Phiên bản

## 1. Document Versioning / Quản lý Phiên bản Tài liệu
- Use Git tags to manage curriculum versions (e.g., `v1.0.0-keycloak22`).
- Dùng Git tags để quản lý các phiên bản giáo trình (ví dụ: `v1.0.0-keycloak22`).
- A `CHANGELOG.md` must be maintained at the root of the repository to track updates.
- Một tệp `CHANGELOG.md` phải được duy trì ở gốc của kho lưu trữ để theo dõi các bản cập nhật.

## 2. Software Version Anchoring / Cố định Phiên bản Phần mềm
- Always specify the exact software versions used for a specific curriculum release to prevent breaking changes for learners.
- Luôn chỉ định chính xác các phiên bản phần mềm được sử dụng cho một bản phát hành giáo trình cụ thể để ngăn chặn các thay đổi gây lỗi cho người học.
