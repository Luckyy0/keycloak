# References Specification
# Đặc tả Tài liệu Tham khảo

## 1. Official Sources / Nguồn Chính thức
- All theoretical explanations MUST link back to the official Keycloak Documentation, relevant RFCs (e.g., RFC 6749 for OAuth2), or OWASP guidelines.
- Toàn bộ giải thích lý thuyết PHẢI liên kết ngược lại Tài liệu Keycloak chính thức, các RFC liên quan (ví dụ: RFC 6749 cho OAuth2) hoặc các hướng dẫn OWASP.

## 2. Formatting / Định dạng
- Provide links in Markdown format at the bottom of each lesson under a `## References` section.
- Cung cấp các liên kết ở định dạng Markdown ở cuối mỗi bài học dưới phần `## References`.
