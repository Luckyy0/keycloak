# Progress Tracking Specification
# Đặc tả Theo dõi Tiến độ

## 1. Checklist Format / Định dạng Danh sách kiểm tra
- Maintain a `MASTER-CHECKLIST.md` in the root of the repository to track which chapters, modules, and lessons have been generated.
- Duy trì tệp `MASTER-CHECKLIST.md` ở gốc của kho lưu trữ để theo dõi các chương, mô-đun và bài học nào đã được sinh ra.
- Use Markdown task lists (`- [ ]`, `- [x]`).
- Dùng danh sách tác vụ Markdown (`- [ ]`, `- [x]`).

## 2. Status Updates / Cập nhật Trạng thái
- The AI must update the tracking file after every successful generation step.
- AI phải cập nhật tệp theo dõi sau mỗi bước sinh thành công.
