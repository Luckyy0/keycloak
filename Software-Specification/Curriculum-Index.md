# Curriculum Index Specification
# Đặc tả Chỉ mục Giáo trình

## 1. Centralized Routing / Định tuyến Tập trung
- This file will act as the master table of contents.
- Tệp này sẽ đóng vai trò là mục lục chính.
- Every generated lesson, lab, and project must be linked here.
- Mọi bài học, bài thực hành và dự án được sinh ra đều phải được liên kết tại đây.

## 2. Dynamic Updates / Cập nhật Động
- The AI must update this index whenever a new file is added to the curriculum.
- AI phải cập nhật chỉ mục này bất cứ khi nào một tệp mới được thêm vào giáo trình.
