# FAQ Specification
# Đặc tả Câu hỏi Thường gặp

## 1. Common Roadblocks / Các Trở ngại Phổ biến
- Every chapter must include a FAQ section addressing common configuration errors (e.g., "Why am I getting Invalid Redirect URI?").
- Mỗi chương phải bao gồm phần Câu hỏi thường gặp giải quyết các lỗi cấu hình phổ biến (ví dụ: "Tại sao tôi nhận được lỗi Invalid Redirect URI?").

## 2. Q&A Format / Định dạng Hỏi & Đáp
- Format as:
  **Q: [Question]**
  **A:** [Detailed Answer]
- Định dạng như sau:
  **Q: [Câu hỏi]**
  **A:** [Câu trả lời chi tiết]
