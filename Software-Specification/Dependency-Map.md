# Dependency Map Specification
# Đặc tả Bản đồ Phụ thuộc

## 1. Chapter Dependencies / Phụ thuộc giữa các Chương
- Certain advanced chapters (e.g., Token Exchange, UMA) require strict prerequisites.
- Các chương nâng cao nhất định (ví dụ: Trao đổi Token, UMA) yêu cầu các điều kiện tiên quyết nghiêm ngặt.
- This document maps which chapters must be completed before attempting another.
- Tài liệu này lập bản đồ các chương nào phải hoàn thành trước khi thử học chương khác.

## 2. Visual Map / Bản đồ Hình ảnh
- Provide a Mermaid flowchart showing the dependency graph of the curriculum.
- Cung cấp một lưu đồ Mermaid hiển thị biểu đồ phụ thuộc của giáo trình.
